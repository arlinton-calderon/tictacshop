TicTacShop requiere de [Python](https://www.python.org/downloads/release/python-380/) v3.8+ para ejecutarse.

Para Windows.

```
cd tictacshop
python3 -m venv venv
./venv/Source/activate
pip install -r requirements.txt
```

Para Linux.

```
cd tictacshop
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```
